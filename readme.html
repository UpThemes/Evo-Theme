Getting Started with Evo

Evo is a flexible, beautiful showcase-style WordPress theme with a very flexible layout, the ability to change thumbnail sizes, post videos, and more!

Skip to Installation Instructions

Features

Evo offers some robust but easy-to-use features:

  • Three Beautiful Color Schemes

    In addition to the default light theme, there are two additional color schemes as well, which can be enabled on the Evo Colors & Images Options page.

  • User Submission Page

    By popular demand, Evo includes a submission page that can be customized to fit your needs inside the UpThemes Framework on the Submission Form options page.

  • AJAX Page Loading

    Evo uses a custom twitter-style pagination to allow you to continue loading new posts into the page without being taken to a new page.

  • WordPress 3.0 Menu Support

    Custom menus allow you to configure your navigation the way you want it! Evo features two menu zones in the header and footer, as well as unlimited widget menus.

  • Simple Video Embedding

    Easy video embedding using the Simple Video Embedder plugin from Press75.com makes video embedding easier than ever. Requires installation of Simple Video Embedder plugin, download separately.

  • Resizable Thumbnails using WordPress 3's Featured Image

    Thumbnail sizes are completely under your control! Just set your desired image sizes in 'Settings > Media' and setting the medium size image to the desired width and a height of 9999 (for images of various heights).

  • Ad Zones Built-in

    There are now two ad zones that accept plain HTML and Javascript below the header and above the footer.

Installing Evo

  1. In WordPress' admin panel, go to 'Themes > Install Themes > Upload ' and browse to the Evo ZIP file.
  2. Click the 'Install Now' button.
  3. Once installed, visit the UpThemes Framework theme options page and set up the various options you require.
  4. After that's done, go to Settings > Reading and set "Blog pages to show at most" to a number that makes your images end at the end of the last row of thumbails. For three across, I recommend 12 or 15, for 4 across, 12 or 16.

Additional Instructions

Adding Gallery Items

To add gallery items, all you need to do is create a new post, upload an image of any size, select a category, and publish the post. DO NOT insert the image into the body of your post. This will result in an image on the right side of the page. If this is what you want, then ignore this warning. To create a lightbox image slider for the post, simply add multiple images to the post. Again, you do not need to post the image in the body of your post for Evo to do its thing.

Using Widgets

  1. Go to Widgets.
  2. On the right side, there will be a set of widget zones for Index, Single, and Page. This allows you to add widgets to different sections of your site
  3. From the list of widgets, select a Text widget and drag it over to the zone you want the ads to be shown in.
  4. Enter the ad code into the textbox and save it.

Adding Advertisements to your Site

Using the Built-In Header and Footer ad zones (recommended)

  1. Go to the UpThemes Framework Ad Manager.
  2. Enter the HTML or Javascript from your ad company into the Ads Below Header or Ads Below Footer area.
  3. Click Save changes.

Using Widgets

  1. Go to Widgets.
  2. On the right side, there will be a set of widget zones for Index, Single, and Page. This allows you to add widgets to different sections of your site
  3. From the list of widgets, select a Text widget and drag it over to the zone you want the ads to be shown in.
  4. Enter the ad code into the textbox and save it.

Adding Widgets to the Footer

  1. In your WordPress Admin, click Widgets on the sidebar.
  2. From the drop-down box under 'Current Widgets' select '1st Subsidiary Aside' and click 'Show.'
  3. Drag the widgets you'd like to see shown in this area from left to right.
  4. Click 'Save.'
  5. You should now see your widgets in the footer.

Using Videos in Posts

Evo works with the Simple Video Embedder plugin from Press75.com.

  1. Install and activate the Simple Video Embedder plugin.
  2. Go to the Video Options (on the left under Settings) page and set them to a default width of 500 and height of 306.
  3. On the post page, simply enter the URL of the video into the Video URL box (in the Post Video Options widget).
  4. Click Update Post.
